CyberTracker Overview

What is CyberTracker

CyberTracker is a universal data collection toolkit. It provides a toolkit for building mobile data capture applications. It provides management of sightings, gps timer tracks, maps, server side communication and numerous UI controls for building custom applications.
Note: CyberTracker applications are written in QML, which is a bit like HTML5 + JavaScript rolled into one. Simple applications don't require any coding.
CyberTracker extends Qt with a number of controls and a data capture engine, which abstracts record creation, navigation and other commonly required features.
